# Open Data Catalog (MVP)

This catalog lists realistic public datasets for a Saskatchewan conservation-prioritization workflow.  
For this MVP, synthetic sample data is generated by script for reproducibility; real data can replace it using the same schema.

| Layer | Purpose | Candidate Source | Format | Typical Update Cadence | License/Use Notes |

## Reporting Context
- Saskatchewan PCAN context: https://www.saskatchewan.ca/business/agriculture-natural-resources-and-industry/land-management/protected-and-conserved-areas-network
- Canada conserved-area reporting context: CPCAD workflows for transparent annual tracking.

## Data Governance Notes
- Capture source URL, access date, download checksum, and schema version in `data/raw/manifest.csv`.
- Keep raw files immutable; all transformations should write to `data/processed/`.
- Document assumptions where public data is incomplete (for example, disturbance proxy substitutions).
